## Implementation Details

### Local Test Mode Logic
```python
if args.local_test:
    nproc = 1
    timeout = 60
else:
    nproc = args.nproc
    timeout = args.timeout
```

### Data Validation
```python
if not data_path.exists():
    print(f"ERROR: Data path does not exist: {data_path}")
    return
```

### Roundtrip Detection
```python
roundtrip_pattern = r"(final_.*?_roundtrip.*)"
roundtrip_match = re.search(roundtrip_pattern, roundtrip_logs)
if roundtrip_match:
    print(f"First detected final_*_roundtrip line:")
    print(f"  {roundtrip_match.group(1)[:120]}")
```
